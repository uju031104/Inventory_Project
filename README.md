# Inventory_Project

프로젝트에 대한 간단한 설명   

<br/>

### 사용한 클래스와 설명
***
<br/>

**Characte 클래스**   

ActorComponent와 UserWidget를 연걸


<br/>


**ActorComponent 클래스** 

인벤토리 정보를 담고 사용

<br/>

**UserWidget 클래스** 

인벤토리 UI와 아이템 UI


<br/>

### 구현한 기능들
***

<br/>

가방은 TArray를 통해서 만드세요

TMap은 저희 TArray에 담긴 아이템들을 확인하게 되면 그 아이템 정보를 Key값으로 아이템 정보들이 뜨도록 만들어주시세요.

TSet은 칭호 획득을 사용하시면됩니다.

해당 칭호가 있어야 아이템을 사용할 수 있도록 만들어주세요.

**인벤토리**   
>TArray안에 아이템을 담는 형식

<br/>

**아이템**   
>구조체로 아이템의 내용을 담음   
      

<br/>

**아이템 정보**   
>TMap으로 <아이템이름, 아이템정보>를 담음

<br/>

**칭호**   
> TSet으로 내가 가지고 있는 칭호 담음   
> TMap으로 <아이템이름, 필요한 칭호>를 담음   

<br/>

**아이템 사용**
>위 두 값을 비교해서 아이템 사용 유무를 결정   
>아이템 사용 시 개수가 차감

<br/>

**UI**   
>인벤토리 UI와 아이템 UI를 사용   
>아이템 UI는 버튼 형식이고 아이템 이미지와 현재 수량을 보여줌   
>이미지 위에 마우스를 대면 아이템 이름과 아이템 정보가 보임   
>아이템 클릭 시 사용 유무에 따라 수량이 차감   
>수량이 0이 되면 UI에서 사라짐
